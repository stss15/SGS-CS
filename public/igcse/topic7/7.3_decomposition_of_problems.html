De-coding Exercise

INPUT Age
IF Age >= 18 THEN
  OUTPUT "Adult"
ELSE
  OUTPUT "Child"
ENDIF

Explanation

"The program asks for the user's age. If the age is 18 or more, it prints 'Adult'. Otherwise, it prints 'Child'."

You Try It

Explain this Algorithm

Total <- 0
FOR K <- 1 TO 5
  INPUT Number
  Total <- Total + Number
NEXT K
OUTPUT Total / 5

Hint: Look at the loop and the final calculation.

Reveal Answer

The Answer

"This algorithm inputs 5 numbers, adds them all up to find the total, and then outputs the average (by dividing the total by 5)."

Purpose: Calculating the average of 5 numbers.

True or False?

Misconception Check

  1. An algorithm must always be written in Python.
  2. Explaining an algorithm means reading the code line by line.
  3. Inputs are what the computer gives back to you.

Press Down to reveal answers

Question 1

An algorithm must always be written in Python.

Answer: False

Algorithms can be written in Pseudocode, Flowcharts, or even plain English. They are language-independent.

Question 2

Explaining an algorithm means reading the code line by line.

Answer: False

It means explaining the purpose and logic in plain English, not just reading the syntax.

Question 3

Inputs are what the computer gives back to you.

Answer: False

Inputs go in. Outputs come out.
