Types of Errors

Grammar vs Meaning

Syntax Error

"Me Tarzan, You Jane"

Bad grammar. The computer doesn't understand.

Program Crashes.

Logic Error

"Turn Left" (when you meant Right)

Perfect grammar. Wrong instruction.

Program Runs (but fails).

Syntax Errors

Breaking the Rules

Every language has rules. If you break them, the code won't run.

Bug Hunter: Level 2

Goal: Average of A and B.

Average <- A + B / 2

Error: Order of Operations (BODMAS).

Fix: (A + B) / 2

Goal: Loop 10 times.

FOR K <- 1 TO 100
  OUTPUT K

Error: Wrong loop limit.

Fix: 1 TO 10

IF Age > 18 THEN
  OUTPUT "Adult"

True or False?

Misconception Check

  1. A Syntax Error allows the program to run but gives wrong results.
  2. Trace Tables are useful for finding Logic Errors.
  3. Misspelling a variable name is a Logic Error.

Press Down to reveal answers

Question 1

A Syntax Error allows the program to run but gives wrong results.

Answer: False

That describes a Logic Error. A Syntax Error prevents the program from running at all.

Question 2

Trace Tables are useful for finding Logic Errors.

Answer: True

By stepping through manually, you can see exactly where the values go wrong.

Question 3

Misspelling a variable name is a Logic Error.

Answer: False

It is usually a Syntax Error (Undeclared Identifier), unless you accidentally spell another valid variable's name!
