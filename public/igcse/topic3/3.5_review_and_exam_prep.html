Key Terms Rapid Fire

Quick recall: Do you know these key terms?

Bus

Wires that transfer data, addresses, and control signals.

Like highways for data.

Opcode & Operand

Opcode: What to do (Instruction)

Operand: What to do it to (Data/Address)

Like verb and noun in a sentence.

Embedded System

A computer built into a device with a dedicated function.

Like a computer hidden in a washing machine.

Latency

The time delay for the read/write head to find data on an HDD.

Like waiting for a record to spin to the right song.

Thrashing

When the HDD is overworked moving data in/out of Virtual Memory.

Like constantly moving files back and forth - very slow!

Exam Question 1: Screens

"Give three advantages of OLED screens compared to LED/LCD." (3 marks)

Strategy: Think about what makes OLED different - no backlight, pixels turn off, etc.

Deep Dive: The Answer

Mark Scheme

Any 3 of these will get you full marks:

  • Thinner and lighter (No backlight needed)
  • Better contrast / True black (Pixels turn off completely)
  • Can be flexible or curved
  • Lower power consumption (sometimes - when showing dark images)
  • Wider viewing angle

Tip: Focus on what OLED can do that LCD can't!

Exam Question 2: Sensors

"A farm monitors fruit growing conditions. Name two sensors and describe how they are used." (4 marks)

Strategy: Think about what conditions affect fruit growth - temperature, moisture, light, etc.

For each sensor, name it AND explain how it's used.

Deep Dive: The Answer

Mark Scheme

You need: Sensor name (1 mark) + How it's used (1 mark) × 2 sensors = 4 marks

Sensor 1: Temperature

How used: Checks if it is too hot or cold

Action: Turn on heater or fans

This is a CONTROL system - it acts!

Sensor 2: Moisture

How used: Checks water level in soil

Action: Turn on sprinklers/irrigation

Again, CONTROL - it responds!

Also valid: Light sensor (for automated blinds/lights), pH sensor (for soil acidity)

Any sensor that makes sense for fruit growing!

Exam Question 3: Storage

"A games console uses both RAM and ROM. Explain why ROM is used for the game program and why RAM is used while playing." (4 marks)

Strategy: Think about the properties of each - ROM = permanent, RAM = fast and changeable.

You need to explain BOTH parts of the question!

Deep Dive: The Answer

Mark Scheme

You need: Why ROM (2 marks) + Why RAM (2 marks) = 4 marks

ROM (Read Only)

Why for game program:

  • Stores the game code permanently
  • Cannot be accidentally deleted or changed
  • Available immediately on startup

ROM = Permanent instructions

RAM (Read/Write)

Why while playing:

  • Stores variable data that changes
  • e.g., Player score, position, health
  • Fast access for smooth gameplay

RAM = Fast, changeable workspace

Exam Question 4: Networks

"Describe three differences between MAC and IP addresses." (3 marks)

Strategy: Think about: Physical vs Logical, Who assigns them, Do they change, Format

You need THREE clear differences (1 mark each).

Deep Dive: The Answer

Mark Scheme

Any 3 of these will get you full marks (1 mark each):

  • Type: MAC is Physical / IP is Logical
  • Assignment: MAC set by Manufacturer / IP set by Network/ISP
  • Changeability: MAC doesn't change / IP can change (Dynamic)
  • Format: MAC is Hexadecimal / IP is Denary (IPv4) or Hex (IPv6)

Tip: Always compare both sides - MAC vs IP!
