The Big Question

The Postman

How does a letter get to the right house in the right country?

You need the right address!

The Network

You need an address (IP) and a door (MAC).

Two addresses for two different purposes.

Key Idea: Networks need addresses to route data. IP = where (logical), MAC = who (physical).

NIC (Network Interface Card)

What hardware does a device need to connect to a network?

The Gateway

Hardware needed to connect a device to a network

Can be wired (Ethernet) or wireless (WNIC)

Like a door to the network - you need it to get in!

The Identity

Every NIC has a unique MAC Address burned into it

Like a fingerprint - unique to each device.

NIC and MAC Address

MAC Address

What is the physical "fingerprint" of a device?

Media Access Control - The physical address

Physical Address

Unique to the device hardware

Does NOT change (usually)

Like a serial number - permanent.

Format

Hexadecimal

00:1A:2B:3C:4D:5E

12 hex digits, grouped in pairs.

Deep Dive: Breakdown

MAC vs IP Address

MAC Address Breakdown

What do the numbers in a MAC address mean?

00:1A:2B : 3C:4D:5E

Split in the middle!

First Half

Manufacturer ID

Identifies who made the card (e.g., Apple, Dell)

Like a brand name.

Second Half

Serial Number

Unique to that specific card

Like a unique ID number.

IP Address

What is the "logical" address on a network?

Internet Protocol - The logical address

Like a postal address - tells you where on the network.

Logical Address

Assigned by the network (Router/ISP)

Changes depending on where you are

Like your home address - changes when you move.

Versions

IPv4: 192.168.1.1 (32-bit)

IPv6: 2001:0db8:... (128-bit)

We ran out of IPv4 addresses!

IPv4 vs IPv6

Static vs Dynamic IP

Does your IP address stay the same or change?

Static IP

Permanent (never changes)

Used for Servers, Printers

Easier to find (always at the same address)

Like a permanent home address.

Dynamic IP

Temporary (Leased - changes over time)

Assigned by DHCP (Dynamic Host Configuration Protocol)

Used for home devices (phones, laptops)

Like a hotel room - you get assigned one when you arrive.

Dynamic vs Static IP

Routers

How does data get from your home network to the internet?

The Traffic Cop

Connects different networks together

e.g., your LAN to the Internet/WAN

Like a bridge between your home and the internet.

Functions

  • Routes data: Directs packets to their destination
  • Assigns IPs: Gives private IP addresses (DHCP)
  • Security: Can act as a firewall

Like a post office - sorts and delivers mail.

Misconception Check

Press Down to reveal answers

Question 1

"A MAC address changes when you move house."

Answer: False

MAC is physical and stays with the device. Your IP address changes.

Question 2

"IPv6 uses 128-bit addresses."

Answer: True

IPv4 uses 32-bit. We switched to IPv6 because we ran out of IPv4 addresses.

Question 3

"A router connects a LAN to the Internet."

Answer: True

It routes traffic between the Local Area Network and the Wide Area Network.

Question 4

"You can have the same public IP as your neighbour."

Answer: False (Usually)

Public IPs must be unique on the internet. You might share one if you are on the same WiFi.
