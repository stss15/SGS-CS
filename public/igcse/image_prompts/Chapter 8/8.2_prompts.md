# 8.2 Arrays - Revision Artifact Prompts

## Concept: One-Dimensional Array
- **Type**: Visual Analogy
- **Description**: Visualizing a list as indexed storage.
- **Prompt**: Create a visual for a "1D Array" (List).
  - **Visual**: A row of lockers or mailboxes.
  - **Labels**:
    - **Name**: "StudentNames".
    - **Indices**: Numbered 0, 1, 2, 3, 4...
    - **Contents**: "Ali", "Ben", "Cat", "Dan"...
  - **Access**: Arrow pointing to Index 2. Text: `StudentNames[2] = "Cat"`.

## Concept: Two-Dimensional Array
- **Type**: Visual Analogy
- **Description**: Visualizing a grid/table.
- **Prompt**: Create a visual for a "2D Array" (Grid).
  - **Visual**: A grid of boxes (Rows and Columns).
  - **Labels**:
    - **Name**: "SeatingPlan".
    - **Rows**: 0, 1, 2 (Vertical axis).
    - **Columns**: 0, 1, 2, 3 (Horizontal axis).
  - **Access**: Arrow pointing to Row 1, Column 3. Text: `SeatingPlan[1, 3]`.
  - **Looping**: Show two arrows (Outer Loop for Rows, Inner Loop for Columns).
