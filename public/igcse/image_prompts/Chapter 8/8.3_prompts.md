# 8.3 File Handling - Revision Artifact Prompts

## Concept: File Operations
- **Type**: Process Flowchart
- **Description**: The sequence of using a file in a program.
- **Prompt**: Create a flowchart for "Using a Text File".
  - **Step 1: Open**: Icon of a folder opening. Text: `OPEN "MyFile.txt" FOR READ/WRITE`.
  - **Step 2: Process**:
    - **Read**: Icon of glasses. Text: `READLINE(File, Line)`.
    - **Write**: Icon of a pencil. Text: `WRITEFILE(File, Line)`.
  - **Step 3: Close**: Icon of a folder closing. Text: `CLOSEFILE("MyFile.txt")`.
  - **Warning**: Alert icon. Text: "Always CLOSE the file to save changes and free memory!"
