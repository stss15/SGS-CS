# 4.1 Types of Software and Interrupts - Revision Artifact Prompts

## Concept: System vs Application Software
- **Type**: Hierarchy Diagram
- **Description**: Visualizing the layers of software.
- **Prompt**: Create a "Software Hierarchy" pyramid.
  - **Top Layer**: "Application Software". Text: "For the User. Word Processor, Games, Browser."
  - **Middle Layer**: "System Software". Text: "For the Computer. OS, Utilities, Drivers, Translators."
  - **Bottom Layer**: "Hardware". Text: "CPU, Memory, Peripherals."
  - **Side Note**: "System Software provides the platform for Applications to run."

## Concept: Operating System Functions
- **Type**: Mind Map / Wheel
- **Description**: The key roles of an OS.
- **Prompt**: Create a central "Operating System" wheel with spokes for each function.
  - **Center**: "Operating System (Manager)".
  - **Spoke 1**: "HCI (Interface)". Text: "GUI (Icons) or CLI (Text). Hides complexity."
  - **Spoke 2**: "Memory Management". Text: "Allocates RAM. Virtual Memory. Paging."
  - **Spoke 3**: "Multitasking". Text: "Time slicing. Prioritizing processes."
  - **Spoke 4**: "Security". Text: "User Accounts. Passwords. Access Rights."
  - **Spoke 5**: "File Management". Text: "Save, Delete, Rename, Folder Structure."
  - **Spoke 6**: "Hardware Management". Text: "Device Drivers. Interrupts. Buffers."

## Concept: Interrupts and Buffers
- **Type**: Flowchart
- **Description**: The process of printing a document using buffers and interrupts.
- **Prompt**: Create a flowchart for "Printing with Interrupts".
  - **Step 1**: "CPU sends data to Printer Buffer."
  - **Step 2**: "CPU moves to other tasks (Multitasking)."
  - **Step 3**: "Printer prints data from Buffer (Slow)."
  - **Step 4**: "Buffer Empty -> Interrupt Signal sent to CPU."
  - **Step 5**: "CPU pauses current task (ISR - Interrupt Service Routine)."
  - **Step 6**: "CPU fills Buffer with more data."
  - **Key**: "Buffer compensates for speed difference. Interrupt tells CPU when to act."

## Concept: Utility Software
- **Type**: Icon Grid
- **Description**: Common utilities and their purpose.
- **Prompt**: Create a grid of "Utility Programs".
  - **Virus Checker**: Shield icon. Text: "Scans for malware. Quarantines files. Heuristic checking."
  - **Defragmentation**: Disk icon with organized blocks. Text: "Reorganizes files to be contiguous. Speeds up access (HDD only)."
  - **Backup**: Cloud/Drive icon. Text: "Copies files to separate media. Restore points."
  - **Screensaver**: Monitor icon. Text: "Prevents screen burn (old monitors). Security (Auto-lock)."
  - **File Compression**: Zipper icon. Text: "Reduces file size. Faster transmission."

## Concept: Firmware (BIOS)
- **Type**: Layered Diagram
- **Description**: Where firmware fits.
- **Prompt**: Create a diagram for "Firmware / BIOS".
  - **Visual**: A chip on a motherboard.
  - **Label**: "BIOS (Basic Input/Output System)".
  - **Text**: "Stored in EEPROM (Non-volatile). Starts up computer (Bootstrap). Checks hardware."
  - **Relationship**: "Hardware <-> Firmware <-> Operating System."
