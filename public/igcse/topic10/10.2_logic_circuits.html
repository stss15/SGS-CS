Quick Recall (10.1)

Q1: What does an AND gate output?

1 only when BOTH inputs are 1

Q2: What does a NOT gate do?

Reverses the input (0→1, 1→0)

Q3: When does XOR output 1?

When inputs are DIFFERENT (one is 0, one is 1)

The Big Question

Can ONE gate solve a real problem?

The Problem

A safe should only open if:

The correct code is entered AND the timer has expired

One Gate?

One AND gate could check both conditions...

But what if there's an override key?

Key Idea: Real problems need multiple gates combined into logic circuits.
What is a logic circuit?

What is a Logic Circuit?

Definition: A logic circuit is a combination of logic gates connected together to solve a specific problem.

Building Blocks

We connect gates like LEGO bricks.

Output of one gate becomes input to another.

Goal

Create a single OUTPUT (0 or 1) based on multiple INPUTS.

Three Ways to Build

Where do we start?

Problem Statement

"The alarm sounds if..."

Logic Expression

X = A AND (NOT B)

Truth Table

All inputs → outputs

The Process: Convert any of these into a logic circuit diagram.

From Problem to Circuit

Real-World Example: Alarm System

"The alarm (X) sounds if the door is open (D) AND the code is NOT entered (C)."

Step 1: Identify Keywords

Door open = D, Code entered = C, Alarm = X

Look for: AND, OR, NOT

Step 2: Write the Logic Expression

X = D AND (NOT C)

Deep Dive: Draw the Circuit

Drawing the Circuit

Expression: X = D AND (NOT C)

Step 1

Identify the inputs:

D (Door) and C (Code)

Step 2

Work inside brackets first:

NOT C

C goes into a NOT gate

Step 3

Connect to the AND gate:

D + (NOT C output) → AND → X

Result: C → NOT gate → output joins D → AND gate → X

From Expression to Circuit

How do we read logic expressions?

Golden Rule: Brackets First!

Just like maths: solve what's inside brackets before anything else.

Example

(A OR B) AND C

1. First do A OR B

2. Then take that result AND it with C

Step-by-Step Build

Step-by-Step Build

Expression: (A OR B) AND C

Gate 1: OR

Inputs: A and B

This handles the brackets first.

Call this output P.

Gate 2: AND

Inputs: P (from OR) and C

This produces final output X.

Circuit Flow: A,B → OR → P → AND (with C) → X

Trace the Signal

Let's test with: A=1, B=0, C=1

Step 1: OR Gate

A(1) OR B(0)

1 OR 0 = ?

P = 1

Step 2: AND Gate

P(1) AND C(1)

1 AND 1 = ?

Result = 1

Final Output

X = 1

The circuit outputs TRUE!

Intermediate Points

What if the circuit has 3+ gates?

Label the Wires!

Between each gate, we label the wire with a letter: P, Q, R, S...

These are called intermediate points.

Why Label?

  • Makes tracing easier
  • Helps find mistakes
  • Essential for building truth tables
Complex Example

Complex Circuit Example

Expression: (A AND B) OR (B NOR C)

Part 1: P

P = A AND B

Inputs A and B → AND gate → P

Part 2: Q

Q = B NOR C

Inputs B and C → NOR gate → Q

Final: X

X = P OR Q

Intermediate outputs P and Q → OR gate → X

Tracing with Intermediate Points

Test: A=1, B=1, C=0

A B C P (A AND B) Q (B NOR C) X (P OR Q)
1 1 0 1 0 1

Worked: P = 1 AND 1 = 1 | Q = 1 NOR 0 = 0 | X = 1 OR 0 = 1

Writing Expressions from Circuits

What if we have the circuit but not the expression?

Work Backwards!

  1. Start with gates connected to inputs
  2. Write expressions for those gates
  3. Join them using the connecting gate
See Example

Worked Example

Imagine a circuit with:

Gate 1: A AND B | Gate 2: B OR C | Gate 3: XOR connects Gates 1 & 2

Step 1

Gate 1: (A AND B)

Gate 2: (B OR C)

Step 2

Join with Gate 3 (XOR):

(A AND B) XOR (B OR C)

Final Expression: X = (A AND B) XOR (B OR C)

Common Mistakes

❌ Ignoring Brackets

NOT C AND D

This is different from:

NOT (C AND D)

First applies NOT to C only. Second applies NOT to the result of C AND D.

❌ Wrong Gate Order

Always work inside brackets first!

Think: BIDMAS but for logic.

Brackets → NOT → AND → OR

Question 1

"We should draw gates in any order."

Answer: FALSE

You must follow brackets and order of operations. Gates inside brackets come first!

Question 2

"Intermediate points help us trace circuits."

Answer: TRUE

Labeling wires (P, Q, R) makes it much easier to find and fix mistakes.

Question 3

"A logic circuit can only have 2 inputs."

Answer: FALSE

Circuits can have many inputs (A, B, C, D, E...). Each gate has max 2 inputs, but the whole circuit can have more.

Question 4

NOT C AND D is the same as NOT (C AND D)

Answer: FALSE

NOT C AND D applies NOT only to C, then ANDs with D.

NOT (C AND D) applies NOT to the entire AND result.
