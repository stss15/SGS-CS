The Big Question

How do computers "think"?

Computers are just metal and silicon. They don't have brains.

Yet they can play chess, recognize faces, and drive cars!

The secret: Billions of tiny electronic switches called logic gates.

By combining simple ON/OFF decisions, we can do anything!

What is a logic gate?

What is a Logic Gate?

Definition: A logic gate is an electronic circuit that takes binary inputs (0 or 1) and produces a binary output.

Inputs

One or more binary values (0 or 1)

Output

A single binary value (0 or 1)

Think of it as: A decision-maker that follows simple rules.

The 6 Logic Gates

NOT

Reverses the input

AND

Both must be 1

OR

Either can be 1

NAND

NOT AND

NOR

NOT OR

XOR

Either but not both

The NOT Gate

The simplest gate: it reverses the input.

Analogy: An opposite day machine.

If you say YES (1), it says NO (0). If you say NO (0), it says YES (1).

Expression: X = NOT A

It's the only gate with just ONE input!

The AND Gate

Output is 1 only if BOTH inputs are 1.

Analogy: A strict parent who says:

"You can go out IF you've done your homework AND tidied your room."

Both must be true!

Expression: X = A AND B

The OR Gate

Output is 1 if EITHER input (or both) is 1.

Analogy: A relaxed parent who says:

"You can go out IF you've done your homework OR tidied your room."

Either one (or both) is enough!

Expression: X = A OR B

The XOR Gate

Output is 1 if inputs are DIFFERENT.

Analogy: A two-way light switch.

If both switches are the same position → light OFF.

If switches are different positions → light ON.

Expression: X = A XOR B

XOR = "Exclusive OR" – one or the other, but not both.

True or False?

Check Your Understanding

  1. A NOT gate has two inputs.
  2. An AND gate outputs 1 if any input is 1.
  3. XOR outputs 1 if the inputs are different.

Press Down to reveal answers

Question 1

A NOT gate has two inputs.

Answer: FALSE

NOT is the only gate with just one input. It simply reverses it.

Question 2

An AND gate outputs 1 if any input is 1.

Answer: FALSE

AND needs BOTH inputs to be 1. That's OR you're thinking of!

Question 3

XOR outputs 1 if the inputs are different.

Answer: TRUE

XOR = "eXclusive OR". It outputs 1 when A ≠ B (different inputs).
