# 10.3 Truth Tables Slide Plan

## 1. Title Slide
- **Slide Type**: Title
- **Content**:
    - Main Title: 10.3 Truth Tables
    - Sub-title: Topic 10: Boolean Logic | Tracing the Flow
- **Visuals**: Standard title slide layout.

## 2. Hook
- **Slide Type**: Concept
- **Question**: How do we test a circuit?
- **Content**:
    - **The Mystery**: We built a circuit, but does it work for *every* case?
    - **The Reality**: We use a **Truth Table** to map every possible input to an output.
    - **Key Idea**: It's the ultimate proof of logic.
- **Visuals**:
    - Box Blue: The Mystery.
    - Box: The Reality.
    - Fragment: Key Idea.

## 3. Counting Rows
- **Slide Type**: Concept
- **Question**: How big is the table?
- **Content**:
    - The number of rows depends on the inputs.
    - **2 Inputs** (A, B): $2^2 = 4$ rows.
    - **3 Inputs** (A, B, C): $2^3 = 8$ rows.
    - **4 Inputs**: $2^4 = 16$ rows.
- **Visuals**:
    - Table showing binary counting (00, 01, 10, 11).

## 4. Tracing a Circuit
- **Slide Type**: Concept
- **Question**: How do we fill it in?
- **Content**:
    - Don't try to do it all in your head!
    - **Strategy**: Label the intermediate points (P, Q).
    - Calculate P first, then Q, then the Final Output.
- **Visuals**:
    - Deep Dive arrow: "Worked Example".

### Vertical: Worked Example (Part 1)
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Circuit**: `(A AND B) OR C`
    - **Step 1**: Calculate `P = A AND B`.
    - Show column P filling up based on A and B.

### Vertical: Worked Example (Part 2)
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Step 2**: Calculate `Output = P OR C`.
    - Show final column filling up based on P and C.

## 5. From Truth Table to Circuit
- **Slide Type**: Concept
- **Question**: Can we go backwards?
- **Content**:
    - Sometimes we have the requirements (Truth Table) and need to build the circuit.
    - **Strategy**: Look at where the output is **1**.
    - Write a logic statement for those rows and join them with OR.
- **Visuals**:
    - Deep Dive arrow: "Logic Synthesis".

### Vertical: Logic Synthesis
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Example**: Output is 1 only when A=1 and B=0.
    - **Expression**: `A AND (NOT B)`.
    - This is how we design chips!

## 6. Plenary
- **Slide Type**: Plenary (True or False)
