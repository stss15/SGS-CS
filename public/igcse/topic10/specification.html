Topic 10: Boolean Logic

Complete Specification & Learning Guide

Cambridge IGCSE Computer Science 0478 (2026–2028)

The Brain of the Computer

At the deepest level, computers don't "think"—they just open and close billions of tiny electronic switches. Boolean Logic is the mathematical system that governs these switches. In this topic, you will learn how to control the flow of 1s and 0s using Logic Gates, creating circuits that can make decisions, do maths, and store data.

What You Will Learn

This topic covers the theoretical foundation of computer decision making:

  • Logic Gates: Identifying and defining the six standard gates: NOT, AND, OR, NAND, NOR, and XOR.
  • Logic Circuits: Combining gates to solve specific problems or represent logic expressions.
  • Truth Tables: Creating tables that map every possible input combination (1s and 0s) to a final output.
  • Logic Expressions: Writing mathematical statements (e.g., A AND B) to represent circuits and problems.

10.1 Standard Logic Gates

What you need to know:

  • Identification: Identify and use the standard symbols for logic gates.
  • Definition: Define and understand the functions of the standard logic gates:
    • NOT
    • AND
    • OR
    • NAND
    • NOR
    • XOR (EOR)

Notes & Guidance

  • Inputs:
    • NOT is a single input gate.
    • All other gates (AND, OR, NAND, NOR, XOR) are limited to two inputs for this syllabus.
  • XOR (Exclusive OR): Outputs 1 only if the inputs are different (one is 1, the other is 0). If both are 1, it outputs 0.
  • NAND/NOR: These are the "Not AND" and "Not OR" gates. They output the exact opposite of their standard counterparts.

10.2 Logic Circuits & Expressions

What you need to know:

  • Creating Circuits: Use logic gates to create logic circuits from:
    • A given problem statement.
    • A logic expression.
    • A truth table.
  • Writing Expressions: Write a logic expression from:
    • A given problem statement.
    • A logic circuit.
    • A truth table.

Notes & Guidance

  • Complexity Limit: Logic circuits will be limited to a maximum of three inputs and one output.
  • Drawing Circuits: Circuits must be drawn clearly for the statement given, without simplification. Lines should join components directly.
  • Expression Format: Use standard notation, e.g., (A AND B) OR (NOT C). Brackets are crucial to show which operations happen first.

10.3 Truth Tables

What you need to know:

  • Completion: Complete a truth table from:
    • A given problem statement.
    • A logic expression.
    • A logic circuit.
  • Binary Output: Produce the binary output from all the possible binary inputs.

Notes & Guidance

  • Rows: The number of rows in a truth table is $2^n$, where $n$ is the number of inputs.
    • 2 Inputs = 4 Rows (00, 01, 10, 11).
    • 3 Inputs = 8 Rows (000 to 111).
  • Intermediate Columns: When solving complex circuits, it is helpful to add extra columns for the intermediate steps (e.g., the output of the first AND gate) before calculating the final output.

Exam Expectations: What You Need to Do

For the Cambridge IGCSE (0478) exam, you must be able to:

  • Draw Accurately: Draw logic circuits cleanly. Ensure NOT gates are triangles with a small circle, and AND/OR gates are distinct (D-shape vs curved).
  • Trace Inputs: Given a circuit and a set of inputs (e.g., A=1, B=0, C=1), trace the 1s and 0s through the gates to find the final output.
  • Translate: Convert fluidly between the three forms: Words ("A is on AND B is off") <-> Circuit Diagram <-> Logic Expression.
  • Fill Tables: Complete a blank column in a truth table based on a provided circuit diagram.
