The Big Question

The Question

Can a computer "think"?

Or is it just following instructions really well?

The Test

The Turing Test: Can a computer fool a human into thinking it is also human?

If yes, does that mean it's "thinking"?

Key Idea: AI is the simulation of intelligent human behaviour. It mimics thinking, but does it actually think?

How Smart Is It?

Are all AIs equally intelligent?

Analogy: Like comparing a calculator (amazing at math) to a human (good at many things).

Narrow AI

Good at ONE thing.

Chess, Spam filter, Siri

Superior to humans in that niche.

Reality: This is what we have today.

General AI

Good at EVERYTHING.

Like a human (can do any task)

Not yet achieved.

Goal: The holy grail of AI research.

Strong AI

SUPERIOR to humans.

Sci-Fi (Terminator, Matrix)

The Singularity.

Reality: Still fiction (for now).

Expert Systems (Old School AI)

How do you capture a doctor's knowledge in a computer?

Uses: Medical diagnosis, Car diagnostics, Mineral prospecting

Like having a doctor in a box, but it never forgets.

Deep Dive: How it Works

The Four Components

The Data

  • Knowledge Base: Database of facts (e.g., "Measles causes spots").
  • Rules Base: IF-THEN logic (e.g., "IF fever AND spots THEN Measles").

Like a textbook and a decision tree.

The Logic

  • Inference Engine: The "Brain" that applies rules to facts.
  • User Interface: The questions asked to the user.

Like a doctor asking questions and thinking.

Example in Action:

User: "I have a fever and spots."

Inference Engine: Checks Rules Base. "IF fever AND spots THEN Measles."

Output: "Diagnosis: Measles (90% probability)."

Machine Learning (New School AI)

How does Netflix know what you like?

The Mystery: You watch one sci-fi movie, and suddenly Netflix suggests 10 more.

How does it know?

Expert System

Humans write the rules.

Like teaching by telling.

Machine Learning

Computer finds the rules from data.

Like learning by example.

Deep Dive: Examples

Machine Learning in Action

Spam Filters

Feed it 10,000 spam emails and 10,000 good emails.

It learns patterns: "Winner!" usually = Spam.

Like learning to spot junk mail by seeing lots of examples.

Image Recognition

Feed it 1 million cat photos.

It learns: ears, whiskers, tail shape = Cat.

Like a child learning what a cat looks like by seeing many cats.

Recommendations

"People who bought X also bought Y."

Collaborative filtering - finds patterns in behavior.

Like a friend recommending a movie you might like.

True or False?

Misconception Check

  1. "Narrow AI can perform any task a human can do."
  2. "In an Expert System, the Inference Engine stores the facts."
  3. "Machine Learning requires a human to write every single rule."
  4. "Deep Learning is a type of Machine Learning."

Press Down to reveal answers

Question 1

"Narrow AI can perform any task a human can do."

Answer: False

It is only good at one specific task.

Question 2

"In an Expert System, the Inference Engine stores the facts."

Answer: False

The Knowledge Base stores facts. The Inference Engine does the reasoning.

Question 3

"Machine Learning requires a human to write every single rule."

Answer: False

That's traditional programming. ML learns rules from data.

Question 4

"Deep Learning is a type of Machine Learning."

Answer: True

It uses neural networks to mimic the brain.
