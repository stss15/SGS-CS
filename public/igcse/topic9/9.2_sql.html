How do you find a needle in a haystack?

If the haystack is a database with millions of rows...

You don't search manually – you ask for it!

SQL (Structured Query Language) is the language we use to "talk" to databases.

The Three Magic Words

SELECT

Which columns do you want?

FROM

Which table is the data in?

WHERE

Which rows match your condition?

Build a query step-by-step

Building a Query

Question: Show the names of students in Red house.

SELECT FirstName, Surname
FROM Students
WHERE House = 'Red'

SELECT

FirstName, Surname

(the columns we want)

FROM

Students

(the table name)

WHERE

House = 'Red'

(the filter)

The Result

FirstName Surname
Ali Khan
John Smith

Note: Only shows the columns we asked for, and only rows matching the condition!

SQL Syntax Rules

SELECT *

The asterisk means "all columns".

SELECT * FROM Students

Text in Quotes

Text values need single quotes.

WHERE House = 'Red'

Numbers: No Quotes

Numbers don't need quotes.

WHERE Age = 14

Not Case Sensitive

SELECT = select = SeLeCt

But we write keywords in CAPS by convention.

Multiple Conditions

What if we need more specific filters?

AND

Both conditions must be true.

OR

Either condition can be true.

NOT

The opposite of the condition.

Examples

AND Example

Question: Students in Red house who are 14.

SELECT FirstName, Surname
FROM Students
WHERE House = 'Red' AND Age = 14

Result: Ali Khan, John Smith (both Red AND 14)

OR Example

Question: Students in Red OR Blue house.

SELECT FirstName, Surname
FROM Students
WHERE House = 'Red' OR House = 'Blue'

Result: All 4 students (everyone is in Red or Blue!)

ORDER BY

How do we sort the results?

ASC

Ascending order (A→Z, 0→9)

Default if not specified.

DESC

Descending order (Z→A, 9→0)

See examples

ORDER BY Examples

Sort by surname alphabetically:

SELECT * FROM Students
ORDER BY Surname ASC

Sort by score (highest first):

SELECT * FROM Students
ORDER BY Score DESC

Result: John (91), Mei (88), Ali (85), Sara (72)

COUNT and SUM

What if we want to count or add up values?

COUNT

Counts how many records match.

SUM

Adds up all values in a column.

Examples

Function Examples

How many students in Red house?

SELECT COUNT(StudentID)
FROM Students
WHERE House = 'Red'

Result: 2

What's the total of all scores?

SELECT SUM(Score)
FROM Students

Result: 336 (85 + 72 + 91 + 88)

Key Takeaways

  1. SELECT = columns, FROM = table, WHERE = filter rows
  2. Use * to select all columns
  3. Text needs 'quotes', numbers don't
  4. AND = both true, OR = either true
  5. ORDER BY sorts results (ASC or DESC)
  6. COUNT = how many, SUM = add up

Press Down to reveal answers

Question 1

SELECT * returns all rows.

Answer: FALSE

SELECT * returns all columns. WHERE filters which rows are returned.

Question 2

Numbers in WHERE need quotes.

Answer: FALSE

Only text values need quotes. Numbers are written without quotes: WHERE Age = 14

Question 3

ORDER BY defaults to ascending.

Answer: TRUE

If you don't specify ASC or DESC, it defaults to ASC (A→Z, 0→9).
