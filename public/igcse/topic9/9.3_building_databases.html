CREATE TABLE

How do we make a new table?

We define the field names and their data types.

See the syntax

CREATE TABLE Syntax

CREATE TABLE Students (
    StudentID VARCHAR(10),
    FirstName VARCHAR(50),
    Surname VARCHAR(50),
    Age INT,
    House VARCHAR(20)
)

Breakdown

StudentID = field name

VARCHAR(10) = data type

SQL Data Types

VARCHAR(n) = Text up to n chars

INT = Integer

DATE = Date

BOOLEAN = True/False

Adding the Primary Key

CREATE TABLE Students (
    StudentID VARCHAR(10) PRIMARY KEY,
    FirstName VARCHAR(50),
    Surname VARCHAR(50),
    Age INT,
    House VARCHAR(20)
)

Adding PRIMARY KEY after a field makes it the unique identifier.

The database will enforce uniqueness automatically!

INSERT INTO

How do we add data to the table?

After creating the structure, we add records.

See the syntax

INSERT INTO Syntax

INSERT INTO Students (StudentID, FirstName, Surname, Age, House)
VALUES ('S001', 'Ali', 'Khan', 14, 'Red')

Breakdown

1. Name the table

2. List the fields

3. Provide the VALUES

Remember!

Text needs 'quotes'

Numbers don't need quotes

Values must match the order!

ALTER TABLE

What if we need to change the structure later?

Adding a new column:

ALTER TABLE Students
ADD Email VARCHAR(100)

Removing a column:

ALTER TABLE Students
DROP COLUMN Email

UPDATE & DELETE

How do we change or remove existing data?

See the commands

UPDATE

Change Ali's house to Blue:

UPDATE Students
SET House = 'Blue'
WHERE StudentID = 'S001'

Warning: Without WHERE, it would change every record!

DELETE

Remove student S002:

DELETE FROM Students
WHERE StudentID = 'S002'

Warning: Without WHERE, it would delete every record!

SQL Commands Summary

Command Purpose
CREATE TABLE Make a new table
INSERT INTO Add a new record
ALTER TABLE Change the table structure
UPDATE Change existing data
DELETE Remove records

Key Takeaways

  1. DDL = commands that build/change structure
  2. CREATE TABLE defines fields and data types
  3. INSERT INTO adds new records
  4. UPDATE and DELETE always need WHERE!
  5. VARCHAR = variable text, INT = integer

Question 1

VARCHAR is for variable length text.

Answer: TRUE

VAR-CHAR = Variable Character. CHAR is fixed length.

Question 2

ALTER TABLE is used to add new records.

Answer: FALSE

ALTER TABLE changes the structure (columns). INSERT INTO adds records (data).

Question 3

DELETE without WHERE removes all records.

Answer: TRUE

Without WHERE to filter, it affects all records. Very dangerous!
