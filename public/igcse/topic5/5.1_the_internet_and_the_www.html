The Big Question

The Highway

What is the physical road that connects every city?

The Websites

What are the shops and services on that road?

Key Idea: The Internet is the road. The WWW is what you find on it.

Internet vs. World Wide Web

Are the "Internet" and the "World Wide Web" the same thing?

The Internet

The physical infrastructure.

A worldwide collection of INTERconnected NETworks.

Hardware, cables, routers, satellites.

The WWW

A service that runs ON the internet.

A collection of multimedia web pages accessed via browsers using HTTP/HTTPS.

Deep Dive: The Analogy

Internet vs WWW

Internet = Roads

Physical infrastructure.

Connects everything together.

Without roads, you can't travel.

WWW = Shops on the Road

Services that use the roads.

Websites are like shops.

You need roads to reach shops.

Other services on the Internet: Email, File Transfer (FTP), Video Calls (VoIP).

The WWW is just ONE of many services!

The Protocol

Why is there a padlock on some websites?

Analogy: HTTP is like sending a postcard (anyone can read it).

HTTPS is like sending a letter in a locked box.

Deep Dive: The Difference

HTTP vs HTTPS

HTTP vs HTTPS

HTTP

Hypertext Transfer Protocol

The set of rules for transferring files.

Problem: Data is sent in plain text.

Anyone intercepting can read it.

HTTPS

Secure version.

Uses SSL/TLS to encrypt data.

Even if intercepted, it's unreadable.

Look for the padlock!

When to use HTTPS: Banking, Shopping, Logins, Any personal data.

The Magic Behind the URL

What happens when you type a URL and press Enter?

The Problem: You type "www.google.com" (human-friendly).

But computers need an IP address like "142.250.191.14" (machine-friendly).

Solution: The Domain Name System (DNS) is like a phone book.

It converts names to numbers.

Deep Dive: The DNS Journey

Step 1: The Request

You type: www.google.com

Browser asks: "Where is www.google.com?"

Sends request to Domain Name Server (DNS).

Step 2: The Lookup

DNS searches its database.

Looks up: www.google.com → 142.250.191.14

If it doesn't know, it asks another DNS server.

Step 3: The Return

DNS sends back: 142.250.191.14

"Here's the IP address!"

Step 4: Connect & Render

Browser uses IP address to contact the web server.

Server sends HTML files.

Browser interprets HTML and displays the page.

All of this happens in milliseconds!

The Memory

Why do websites "remember" you?

Analogy: Like a waiter remembering your favorite table.

Websites use Cookies to remember you.

Deep Dive: Types of Cookies

Cookies Explained

Two Types of Cookies

Session Cookies

Temporary

  • Stored in RAM (Memory)
  • Deleted when browser closes
  • Example: Shopping basket

Like a sticky note on your desk.

Persistent Cookies

Permanent

  • Stored on Hard Drive
  • Remain until expiry or deleted
  • Example: "Remember Me" login

Like a note in your diary.

What Are Cookies Used For?

Logins

Storing passwords (Persistent).

Shopping

Shopping basket (Session).

Tracking

Targeted ads.

Preferences

Language, theme.

Analytics

Website usage stats.

Press Down to reveal answers

Question 1

"The Internet and the World Wide Web are the same thing."

Answer: False

The Internet is the physical infrastructure; the WWW is the multimedia pages accessed via browsers.

Question 2

"Session cookies are stored on your hard drive forever."

Answer: False

Session cookies are temporary, stored in memory, and deleted when the browser closes.

Question 3

"DNS converts an IP address into a URL."

Answer: False

It's the other way around: DNS converts a URL (human-readable) into an IP address (computer-readable).
