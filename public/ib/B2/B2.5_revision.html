File Processing

Persistence, I/O Operations, and Data Handling

Core Concepts

Persistence

When a program closes, variables in RAM are lost. Files provide persistence, allowing data to be saved (written) and reloaded (read) later.

File & Buffers

A Text File is a sequence of characters structured in lines. A Buffer is a temporary memory area used to move data between the program and the file system.

File Modes

Mode Description Behavior if file exists Behavior if file doesn't exist
'r' Read Reads from start. Error (File Not Found)
'w' Write OVERWRITES (deletes content). Creates new file.
'a' Append Adds to end. Creates new file.
Reading Files

Using the with open() statement is best practice. It automatically closes the file, preventing resource leaks.

Line by Line (Preferred)

# Efficient for big files
with open("data.txt", "r") as f:
    for line in f:
        print(line.strip())

Read All Lines

# Loads entire file into a list
with open("data.txt", "r") as f:
    lines = f.readlines()
    
# Process list
print(lines[0])
The newline character

Lines in a file end with \n. Use .strip() to remove it when reading strings.

Writing & Appending

Writing (Overwriting)

# CAUTION: Deletes old content!
with open("output.txt", "w") as f:
    f.write("First Line\n")
    f.write("Second Line\n")

Appending

# Safe: Add to end
with open("log.txt", "a") as f:
    f.write("New Log Entry\n")
Processing Parallel Arrays

Often data is stored in separate files (e.g., names.txt and scores.txt) or columns that relate to each other by index.

Scenario: Loading Sales Data

names = []
sales = []

# Assuming a CSV format: "Name,Sales"
with open("sales_data.txt", "r") as file:
    for line in file:
        parts = line.strip().split(",")
        names.append(parts[0])
        sales.append(int(parts[1]))

# Now we can process them together
for i in range(len(names)):
    print(f"{names[i]} sold {sales[i]} units")

Scenario: Finding the Top Seller

max_sales = -1
winner_index = -1

for i in range(len(sales)):
    if sales[i] > max_sales:
        max_sales = sales[i]
        winner_index = i

print(f"Winner: {names[winner_index]}")
Keyword Glossary
Keyword Definition
File Processing The manipulation of data stored in a file (Reading, Writing, Appending).
Persistence Data remaining available in storage after the program terminates.
Buffer Temporary memory used to move data between program and file system.
Text File A structured sequence of lines of electronic text.
Open Mode Permission setting ('r', 'w', 'a') used when opening a file.
Parallel Arrays Two or more arrays of the same size where data at the same index is related.
CSV Comma Separated Values - a common plain text format.
