Programming Fundamentals

Quick Reference Study Guide & Revision Notes

Data Types & Variables

Variables are containers for storing data values. In Python, variables are dynamically typed (you don't declare the type explicitly).

Core Data Types

Type Description Examples
int Whole numbers 10, -5
float Decimal numbers 3.14, 2.0
str Text strings. (Python has no separate 'char' type) "Hello", 'A'
bool Logic values True, False

Variable Swapping (The "Cup" Analogy)

To swap two variables a and b, you need a temporary variable (like a third cup).

a = 5      # Cup A (Water)
b = 10     # Cup B (Tea)

temp = a   # 1. Pour 'a' into empty temp cup
a = b      # 2. Pour 'b' into 'a'
b = temp   # 3. Pour 'temp' into 'b'
Revision Tips
  • Misconception: Don't confuse assignment (=) with equality comparison (==).
  • Exam Tip: If asked to "declare" a variable in pseudocode, Python just assigns a value (e.g., score = 0).
  • Watch Out: "5" is a String, 5 is an Integer. You cannot add them directly!
Operators

Operators perform operations on variables and values. Python has specific symbols for arithmetic, comparison, and logic.

Arithmetic Operators

Operator Description Example
+ Add 5 + 2 → 7
- Subtract 5 - 2 → 3
* Multiply 5 * 2 → 10
/ Float Division (Decimal result) 7 / 2 → 3.5
// Floor Division (Integer result) 7 // 2 → 3
% Modulus (Remainder) 7 % 2 → 1
** Exponentiation (Power) 2 ** 3 → 8

Relational Operators

== Equal to
!= Not equal to
< Less than
> Greater than
<= Less or Equal
>= Greater or Equal

Logical Operators

and Both must be true
or At least one is true
not Invert value (True → False)
Critical Differences
  • / always creates a float. // always rounds down to an int.
  • Short-circuiting: or stops early if the first condition is True.
  • BODMAS/BIDMAS applies to programming too!
String Manipulation

Strings are sequences of characters. They are immutable (cannot be changed in place).

Slicing Syntax: string[start : stop : step]

txt = "Computer"
# Indices: 01234567

print(txt[0:3])   # "Com" (Indices 0, 1, 2 - STOP excluded)
print(txt[3:])    # "puter" (From 3 to end)
print(txt[-1])    # "r" (Last character)

Common Methods

len(s) Returns number of characters (including spaces).
s.strip() Removes whitespace from start and end.
s.replace(old, new) Replaces all occurrences of a substring.
s + s Concatenation (joining strings).
Note

Slicing [a:b] includes a but excludes b. Also, string methods return a new string; they don't modify the original.

Error Handling

Exception handling prevents programs from crashing when runtime errors occur (like dividing by zero).

try:
    num = int(input("Enter number: "))
    result = 100 / num
except ValueError:
    # Runs if user types text instead of a number
    print("That is not a valid number!")
except ZeroDivisionError:
    # Runs if user types 0
    print("Cannot divide by zero!")
except Exception as e:
    # Generic catch-all
    print(f"Unknown error: {e}")
finally:
    # ALWAYS runs (for cleanup)
    print("Execution finished.")
Tips
  • Syntax Errors happen before code runs. Exceptions happen during execution.
  • Always catch specific errors (e.g. ValueError) before generic ones.
Debugging & Trace Tables

Trace tables are manual checks of logic. Breakpoints allow you to pause execution in an IDE.

Algorithm: Linear Search

nums = [10, 50, 30]
target = 50
found = False
for i in range(len(nums)):
    if nums[i] == target:
        found = True
        break

Trace Table

step i nums[i] nums[i]==target found
Init - - - False
Loop 1 0 10 False False
Loop 2 1 50 True True
Break - - - True
