The Locker Problem

You need to store gym kits for 100 students.

Option A: Variables

100 individual lockers scattered around the school.

kit1 = "Alice"
kit2 = "Bob"
# ... ... ...
kit100 = "Zoe"

Chaos. Hard to find.

Option B: Data Structure

One room with 100 numbered lockers.

kits = ["Alice", "Bob", ..., "Zoe"]

Organized. Indexed.

Arrays (1D)

Think of a row of Post Office Boxes.

# Index:  0      1      2      3
prices = [10.50, 5.00, 20.00, 7.50]
  • Identifier: prices (The Address)
  • Index: 0, 1, 2, 3 (Box Number)
  • Element: 10.50 (The Content)

Warning: Computers count from 0!

Tracing a Stack

Follow the operations step-by-step.

s = Stack()
s.push(5)
s.push(8)
s.pop()
s.push(3)

Press Down to Trace

1. push(5)

Stack: [5]

2. push(8)

Stack: [5, 8] (8 is top)

3. pop()

Removes top (8).

Stack: [5]

4. push(3)

Adds 3 to top.

Final Stack: [5, 3]

Tracing a Queue

Linear Queue: Join back, Leave front.

q = Queue()
q.enqueue("A")
q.enqueue("B")
q.dequeue()
q.enqueue("C")

Press Down to Trace

1. enqueue("A")

Queue: [A]

2. enqueue("B")

Queue: [A, B]

3. dequeue()

Removes front ("A").

Queue: [B]

4. enqueue("C")

Adds to back ("C").

Final Queue: [B, C]

Static vs Dynamic

Static (The Bus)

Fixed Seats.

Once built, you can't add more seats. Fast to find your seat (index).

Example: Array

Dynamic (The Train)

Add Carriages.

Can grow as needed. Harder to walk to the back.

Example: Linked List / Python List
