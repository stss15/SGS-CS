Unit 1: Introduction to Computing

Digital Literacy, Organisation & Essential School Tools

Unit Overview

This unit gives you the foundation you need to work confidently and independently on your school devices. You will learn how to navigate the MacBook, organise files properly, and use Office 365 tools as your main learning platform. By the end, you will know where your work lives, how to save it correctly, and how to access the resources teachers share in Teams and OneNote. The focus here is on building good habits early so every future unit runs smoothly.

Unit Objectives

  • Navigate the macOS interface confidently (Finder, Dock, Launchpad, file paths).
  • Create and maintain an organised folder structure for schoolwork.
  • Save, locate, and manage files using both local storage and OneDrive.
  • Access and use Teams as the hub for communication, assignments, and resources.
  • Navigate OneNote Class Notebook (Content Library vs. personal notebook) and create/edit pages.
  • Understand the difference between local and cloud storage and when each is useful.
  • Work collaboratively on shared OneNote pages in a safe and responsible way.

Concepts

This unit focuses on three St George’s Computer Science concepts:

Adaptation

How systems and users adjust to new tools and workflows.

In this unit: Students adapt to the school’s digital ecosystem — learning to move between apps, access cloud resources, and develop routines that support independent learning.

Form

How digital information is organised and structured.

In this unit: Students build consistent file structures and learn how systems like Teams/OneNote organise content.

Function

Understanding the purpose behind tools and systems.

In this unit: Students learn what Teams, OneNote, and cloud storage are for — and why the school uses them.
