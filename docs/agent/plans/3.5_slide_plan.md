### 4. Exam Question 1: Screens
- **Slide:** OLED vs LCD.
- **Question:** "Give three advantages of OLED screens compared to LED/LCD."
- **Deep Dive (Vertical):** The Answer.
    1.  Thinner/Lighter (No backlight).
    2.  Better contrast (True black).
    3.  Flexible/Curved.
    4.  Lower power (sometimes).

### 5. Exam Question 2: Sensors
- **Slide:** The Fruit Farm.
- **Question:** "A farm monitors fruit growing conditions. Name two sensors and how they are used."
- **Deep Dive (Vertical):** The Answer.
    - **Temperature Sensor:** Check if too hot/cold -> Turn on heater/fan.
    - **Moisture Sensor:** Check soil water -> Turn on sprinklers.
    - **Light Sensor:** Check sunlight -> Automated blinds?

### 6. Exam Question 3: Storage
- **Slide:** The Games Console.
- **Question:** "Why use ROM for the game program? Why use RAM?"
- **Deep Dive (Vertical):** The Answer.
    - **ROM:** Permanent, can't be changed/deleted accidentally, ready immediately.
    - **RAM:** To store variable data (score, player position) that changes while playing.

### 7. Exam Question 4: Networks
- **Slide:** IP vs MAC.
- **Question:** "Describe three differences between MAC and IP addresses."
- **Deep Dive (Vertical):** The Answer.
    - MAC is Physical / IP is Logical.
    - MAC is set by Manufacturer / IP set by Network/ISP.
    - MAC doesn't change / IP can change (Dynamic).
    - MAC is Hex / IP is Denary (IPv4) or Hex (IPv6).

    - Don't just memorise, UNDERSTAND.
