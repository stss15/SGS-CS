# Slide Plan: 3.2 Input and Output Devices

## Source Material
- **File:** `3.2.txt`
- **Topic:** Input and Output Devices, Sensors

## Slide Structure

### 1. Title Slide
- **Main Title:** 3.2 Input & Output
- **Sub-Title:** Topic 3: Hardware

### 2. The Big Question (Hook)
- **Concept:** The Senses.
- **Question:** How does a computer see, hear, and speak?
- **Visual:** Eye/Ear icons (Input) vs Mouth/Hand icons (Output).
- **Key Idea:** Input = Data IN. Output = Information OUT.

### 3. Input Devices: Scanners
- **Slide:** Reading the World.
- **Content:**
    - **2D Scanners:** Documents to images (OCR).
    - **3D Scanners:** Objects to 3D models (for CAD/3D Printing).
    - **Barcode Scanners:** Laser reads black/white lines.
    - **QR Codes:** 2D barcodes (more data, readable by phones).
- **Deep Dive (Vertical):** Barcodes vs QR Codes.
    - Comparison table (Capacity, Readability, Error correction).

### 4. Input Devices: Capturing Reality
- **Slide:** Cameras & Audio.
- **Content:**
    - **Digital Cameras:** Light -> Sensor (CCD/CMOS) -> ADC -> Digital Image.
    - **Microphones:** Sound -> Diaphragm -> ADC -> Digital Sound.
- **Key Term:** ADC (Analogue to Digital Converter).

### 5. Input Devices: Interaction
- **Slide:** Touching & Typing.
- **Content:**
    - **Keyboards:** Membrane/Mechanical. RSI risks.
    - **Mice:** Optical (camera + DSP).
    - **Touch Screens:**
        - **Resistive:** Pressure (cheap, stylus).
        - **Capacitive:** Electricity (fingers, multi-touch).
        - **Infra-red:** Light beams (durable).

### 6. Sensors
- **Slide:** The Nervous System.
- **Content:**
    - Read physical data (Temp, Light, Pressure, etc.).
    - Send ANALOGUE data to ADC.
- **Deep Dive (Vertical):** Control vs Monitoring.
    - **Monitoring:** Just watching (e.g., weather station).
    - **Control:** Watching AND acting (e.g., AC unit, greenhouse). Feedback loop.

### 7. Output Devices: Display & Print
- **Slide:** Showing Results.
- **Content:**
    - **Screens:** LCD (Liquid Crystal) vs LED (Backlit) vs OLED (Organic - no backlight, thinner, better black).
    - **Projectors:** DLP (Mirrors) vs LCD.
    - **Printers:**
        - **Inkjet:** Liquid ink (Thermal bubble/Piezo). Good for photos.
        - **Laser:** Toner powder (Static electricity). Fast, text.
        - **3D Printers:** Additive manufacturing (Layers).

### 8. Output Devices: Action
- **Slide:** Moving Things.
- **Content:**
    - **Actuators:** Motors, pumps, switches.
    - **Speakers:** DAC (Digital to Analogue) -> Sound.

### 9. Plenary: True or False?
- **Slide:** Misconception Check.
- **Questions:**
    1.  "A QR code holds less data than a barcode." (False)
    2.  "Capacitive touch screens work with gloves." (False - usually)
    3.  "Sensors send digital data directly to the CPU." (False - need ADC)
    4.  "OLED screens need a backlight." (False)

## Interactive Elements
- **Deep Dives:** Vertical slides for specific tech comparisons (e.g., Resistive vs Capacitive).
- **Icons:** Lots of icons for different devices.
