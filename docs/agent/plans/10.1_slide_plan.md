# 10.1 Logic Gates Slide Plan

## 1. Title Slide
- **Slide Type**: Title
- **Content**:
    - Main Title: 10.1 Logic Gates
    - Sub-title: Topic 10: Boolean Logic | The Building Blocks
- **Visuals**: Standard title slide layout.

## 2. Hook
- **Slide Type**: Concept
- **Question**: How do computers "think"?
- **Content**:
    - **The Mystery**: Computers are just metal and silicon. They don't have brains.
    - **The Reality**: They use billions of tiny digital switches called **Logic Gates**.
    - **Key Idea**: By combining simple ON/OFF decisions, we can perform complex calculations.
- **Visuals**:
    - Box Blue: The Mystery.
    - Box: The Reality.
    - Fragment: Key Idea.

## 3. The Basics
- **Slide Type**: Concept
- **Question**: What is a logic gate?
- **Content**:
    - A component that takes **binary inputs** (0 or 1) and produces a **binary output**.
    - We use **Truth Tables** to show every possible combination.
- **Visuals**:
    - Diagram of a generic gate (Input A, Input B -> Gate -> Output X).
    - Deep Dive arrow: "The 6 Standard Gates".

## 4. The NOT Gate
- **Slide Type**: Vertical Deep Dive
- **Question**: What is the simplest gate?
- **Content**:
    - **NOT Gate** (Inverter).
    - **Function**: Reverses the input. If Input is 1, Output is 0.
    - **Symbol**: Triangle with a circle.
    - **Notation**: X = NOT A (or $\bar{A}$).
- **Visuals**:
    - NOT Gate Symbol.
    - Truth Table (0->1, 1->0).

## 5. AND & OR Gates
- **Slide Type**: Concept
- **Question**: How do we make decisions?
- **Content**:
    - **AND Gate**: Needs **BOTH** inputs to be 1.
    - **OR Gate**: Needs **EITHER** (or both) inputs to be 1.
- **Visuals**:
    - Deep Dive arrow: "See the Truth Tables".

### Vertical: AND Gate
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Symbol**: D-shape.
    - **Logic**: Output is 1 ONLY if A=1 AND B=1.
    - **Notation**: X = A AND B (or A.B).
    - **Truth Table**: 00->0, 01->0, 10->0, 11->1.

### Vertical: OR Gate
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Symbol**: Curved shield shape.
    - **Logic**: Output is 1 if A=1 OR B=1.
    - **Notation**: X = A OR B (or A+B).
    - **Truth Table**: 00->0, 01->1, 10->1, 11->1.

## 6. NAND & NOR Gates
- **Slide Type**: Concept
- **Question**: What is the opposite?
- **Content**:
    - **NAND**: NOT AND. The opposite of AND.
    - **NOR**: NOT OR. The opposite of OR.
- **Visuals**:
    - Deep Dive arrow: "See the Truth Tables".

### Vertical: NAND Gate
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Symbol**: AND gate with a circle at the output.
    - **Logic**: Output is 0 ONLY if A=1 AND B=1. (1 everywhere else).
    - **Notation**: X = A NAND B.
    - **Truth Table**: 00->1, 01->1, 10->1, 11->0.

### Vertical: NOR Gate
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Symbol**: OR gate with a circle at the output.
    - **Logic**: Output is 1 ONLY if A=0 AND B=0. (0 everywhere else).
    - **Notation**: X = A NOR B.
    - **Truth Table**: 00->1, 01->0, 10->0, 11->0.

## 7. The XOR Gate
- **Slide Type**: Concept
- **Question**: What if we want "one or the other, but not both"?
- **Content**:
    - **XOR** (Exclusive OR).
    - **Logic**: Output is 1 if inputs are **different**.
    - **Symbol**: OR gate with a curved line on the back.
    - **Notation**: X = A XOR B.
- **Visuals**:
    - Deep Dive arrow: "See the Truth Table".

### Vertical: XOR Truth Table
- **Slide Type**: Vertical Deep Dive
- **Content**:
    - **Truth Table**:
        - 0, 0 -> 0 (Same)
        - 0, 1 -> 1 (Different)
        - 1, 0 -> 1 (Different)
        - 1, 1 -> 0 (Same)

## 8. Plenary
- **Slide Type**: Plenary (True or False)
- **Content**:
    1. "A NOT gate has two inputs." (False - only 1).
    2. "An AND gate outputs 1 if any input is 1." (False - needs both).
    3. "NAND is the opposite of AND." (True).
    4. "XOR outputs 1 if inputs are the same." (False - outputs 1 if different).
