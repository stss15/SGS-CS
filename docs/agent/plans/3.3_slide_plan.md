# Slide Plan: 3.3 Data Storage

## Source Material
- **File:** `3.3.txt`
- **Topic:** Primary Memory, Secondary Storage, Virtual Memory, Cloud

## Slide Structure

### 1. Title Slide
- **Main Title:** 3.3 Data Storage
- **Sub-Title:** Topic 3: Hardware

### 2. The Big Question (Hook)
- **Concept:** The Desk vs The Filing Cabinet.
- **Question:** Why don't we just store everything on the hard drive?
- **Visual:** Desk (RAM) vs Cabinet (HDD).
- **Key Idea:** Speed vs Capacity.

### 3. Primary Memory
- **Slide:** The Workspace.
- **Content:**
    - Directly accessible by CPU.
    - **RAM (Random Access Memory):** Volatile, Read/Write. The "Desk".
    - **ROM (Read Only Memory):** Non-volatile, Read Only. The "Instructions". (BIOS/Bootstrap).
- **Deep Dive (Vertical):** DRAM vs SRAM.
    - **DRAM:** Needs refresh, cheaper, main RAM.
    - **SRAM:** Fast, no refresh, expensive, Cache.

### 4. Secondary Storage
- **Slide:** The Filing Cabinet.
- **Content:**
    - Non-volatile, Slower, Larger capacity.
    - **Magnetic:** HDD (Platters, Read/Write heads).
    - **Optical:** CD/DVD/Blu-ray (Lasers, Pits/Lands).
    - **Solid State:** SSD/USB (Flash memory, Electrons, Gates).

### 5. Storage Technologies Comparison
- **Slide:** Which one to choose?
- **Content:**
    - **HDD:** Cheap, high capacity, fragile (moving parts).
    - **SSD:** Fast, durable, expensive (per GB).
    - **Optical:** Portable, cheap distribution, slow.

### 6. Virtual Memory
- **Slide:** Faking it.
- **Content:**
    - When RAM is full...
    - Use part of HDD/SSD as "fake RAM".
    - **Swap:** Move pages of data in and out.
    - **Problem:** Disk Thrashing (Slow!).

### 7. Cloud Storage
- **Slide:** The Computer Elsewhere.
- **Content:**
    - Storing data on remote servers (Internet).
    - **Pros:** Access anywhere, auto-backup, easy sharing.
    - **Cons:** Need internet, security risks, monthly cost.

### 8. Plenary: True or False?
- **Slide:** Misconception Check.
- **Questions:**
    1.  "RAM keeps its data when power is off." (False)
    2.  "SSD has moving parts." (False)
    3.  "Virtual memory is faster than RAM." (False)
    4.  "Blu-ray uses a red laser." (False - Blue/Violet)

## Interactive Elements
- **Comparison Tables:** RAM vs ROM, HDD vs SSD.
- **Visuals:** Diagrams of HDD platters vs SSD chips.
