Primary Storage

The CPU cannot access the HDD directly. It's too slow.

1. The Bottleneck
The CPU processes billions of instructions per second. The Hard Drive can only deliver millions.
2. The Solution
We need a "middle-man". Fast memory that sits close to the CPU.

Core Concept

Volatile Memory

Computer memory that requires power to maintain the stored information. If power is lost, the data is lost.

Write this down verbatim. "Volatile" comes from chemistry (evaporates easily).

RAM vs ROM

The two main types of Primary Memory.

RAM

  • Random Access Memory
  • Read & Write
  • Volatile

ROM

  • Read Only Memory
  • Read Only (can't change)
  • Non-Volatile
Deep Dive: Firmware

Deep Dive: Firmware

Why do we need ROM if we can't write to it?

The Bootstrap Loader
When the computer turns on, RAM is empty (volatile). The CPU needs instructions immediately. It looks to the ROM for the BIOS to start the system.

Hinge Question

You are writing a Word document but haven't saved it yet. The power cuts. What happens?

A) It is saved in ROM, so it is safe.
B) It is lost because it was in RAM (Volatile).
C) It is saved to the Hard Drive automatically.

B

It is lost.

Open documents live in RAM. RAM is volatile. No power = No Data.

Buggy Code 

count = 0
while count < 5:
    print(count)
# Loop runs forever
The Fix 

count = 0
while count < 5:
    print(count)
    count = count + 1  # Added increment
