# 7.3 Standard Methods of Solution - Image Prompts

## Concept: Linear Search
- **Type**: Sequence Diagram
- **Description**: Searching through a list.
- **Prompt**: Create a diagram titled "Linear Search". Show a row of boxes with numbers: [5, 2, 9, 1, 7]. Target is "9". Step 1: Arrow points to "5" - Label "Match? No". Step 2: Arrow points to "2" - Label "Match? No". Step 3: Arrow points to "9" - Label "Match? Yes! Found".

## Concept: Bubble Sort
- **Type**: Step-by-Step Diagram
- **Description**: Sorting numbers.
- **Prompt**: Create a diagram titled "Bubble Sort Pass". Show a vertical list: [5, 3, 8, 1]. Step 1: Compare 5 and 3. Swap? Yes. Result: [3, 5, 8, 1]. Step 2: Compare 5 and 8. Swap? No. Step 3: Compare 8 and 1. Swap? Yes. Result: [3, 5, 1, 8]. Label: "Largest value bubbles to top".

## Concept: Totalling and Counting
- **Type**: Comparison Diagram
- **Description**: Difference between totalling and counting.
- **Prompt**: Create a split diagram. Left: "Counting". Show a variable "Count = 0". Show loop: "Count = Count + 1". Visual: Tally marks. Right: "Totalling". Show a variable "Total = 0". Show loop: "Total = Total + Number". Visual: Calculator summing prices.
