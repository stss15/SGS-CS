# 6.1 Automated Systems - Revision Artifact Prompts

## Concept: The Automated System Loop
- **Type**: Cycle Diagram / Flowchart
- **Description**: The fundamental loop of Input-Process-Output in automation.
- **Prompt**: Create a cycle diagram titled "How Automated Systems Work".
  - **Step 1: Input**: Icon of a Sensor (e.g., Thermometer). Label: "Sensors". Text: "Measure physical data (Temp, Light, Pressure)."
  - **Step 2: Conversion**: Arrow to "ADC". Text: "Analogue to Digital Converter. Changes real-world signals to binary."
  - **Step 3: Process**: Icon of a Chip. Label: "Microprocessor". Text: "Compares input data to pre-set values. Decides what to do."
  - **Step 4: Output**: Arrow to "Actuators". Text: "Motors, Pumps, Valves, Heaters."
  - **Step 5: Action**: The physical change (e.g., "Heater turns on").
  - **Feedback Loop**: Arrow going back to Step 1 (Sensors continue to measure).
