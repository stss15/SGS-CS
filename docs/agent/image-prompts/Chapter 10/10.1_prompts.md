# 10.1 Logic Gates - Revision Artifact Prompts

## Concept: Standard Logic Gates
- **Type**: Icon List / Symbol Chart
- **Description**: The six standard logic gates.
- **Prompt**: Create a chart for "Logic Gates".
  - **NOT**: Triangle with a circle. Text: "Inverter. 0->1, 1->0."
  - **AND**: D-shape. Text: "Both inputs must be 1."
  - **OR**: Curved shield shape. Text: "Either input can be 1."
  - **NAND**: AND symbol + circle. Text: "NOT AND. Opposite of AND."
  - **NOR**: OR symbol + circle. Text: "NOT OR. Opposite of OR."
  - **XOR**: OR symbol with double curved line. Text: "Exclusive OR. Inputs must be DIFFERENT."

## Concept: Truth Table Basics
- **Type**: Visual Guide
- **Description**: How to construct a truth table.
- **Prompt**: Create a guide for "Truth Tables".
  - **2 Inputs (A, B)**: Show rows `00`, `01`, `10`, `11`. (4 Rows).
  - **3 Inputs (A, B, C)**: Show rows `000` to `111`. (8 Rows).
  - **Formula**: `2^n` rows (where n = number of inputs).
  - **Columns**: Input columns on left, Output column (X) on right.
