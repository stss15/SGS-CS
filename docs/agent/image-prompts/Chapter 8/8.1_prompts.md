# 8.1 Programming Concepts - Revision Artifact Prompts

## Concept: Variables vs Constants
- **Type**: Comparison Visual / Analogy
- **Description**: Distinguishing between changeable and fixed data stores.
- **Prompt**: Create a comparison of "Variable vs. Constant".
  - **Variable**:
    - Visual: An open cardboard box labeled "Score".
    - Action: Items (numbers) being put in and taken out.
    - Meaning: "Value CAN change while program runs."
  - **Constant**:
    - Visual: A locked metal safe labeled "PI (3.142)".
    - Action: Locked shut.
    - Meaning: "Value CANNOT change while program runs."

## Concept: Data Types
- **Type**: Icon List
- **Description**: The standard data types.
- **Prompt**: Create an icon list for "Data Types".
  - **Integer**: Icon of a whole number (e.g., `42`). Text: "Whole numbers."
  - **Real**: Icon of a decimal number (e.g., `3.14`). Text: "Decimal numbers."
  - **Char**: Icon of a single letter block (e.g., `'A'`). Text: "Single character."
  - **String**: Icon of a word in quotes (e.g., `"Hello"`). Text: "Sequence of characters."
  - **Boolean**: Icon of a Switch (On/Off). Text: "True or False."

## Concept: Selection Structures
- **Type**: Flowchart Comparison
- **Description**: IF vs CASE statements.
- **Prompt**: Create a visual comparison of "Selection Methods".
  - **IF-THEN-ELSE**:
    - Visual: A fork in the road (2 paths).
    - Use: "Checking a condition (True/False)."
  - **CASE (Switch)**:
    - Visual: A roundabout with many exits.
    - Use: "Choosing from multiple specific values."

## Concept: Iteration Structures
- **Type**: Visual Guide
- **Description**: The three types of loops.
- **Prompt**: Create a guide to "Loops / Iteration".
  - **FOR Loop**:
    - Visual: A runner doing exactly 10 laps.
    - Text: "Count-Controlled. Run a set number of times."
  - **WHILE Loop**:
    - Visual: A guard checking a pass BEFORE letting you in.
    - Text: "Pre-Condition. Check first. Might never run."
  - **REPEAT UNTIL Loop**:
    - Visual: Doing a task, then checking if done.
    - Text: "Post-Condition. Run at least once. Check at end."

## Concept: String Handling
- **Type**: Visual Operations
- **Description**: Common string functions.
- **Prompt**: Create a visual for "String Operations" using the string `"Computer"`.
  - **Length**: Ruler measuring the word. Result: `8`.
  - **Substring(0, 3)**: Scissors cutting out `"Com"`.
  - **Upper**: Arrow pointing to `"COMPUTER"`.
  - **Lower**: Arrow pointing to `"computer"`.

## Concept: Operators
- **Type**: Symbol Chart
- **Description**: Arithmetic, Logical, and Boolean operators.
- **Prompt**: Create a chart for "Programming Operators".
  - **Arithmetic**: `+`, `-`, `*`, `/`, `^` (Power), `MOD` (Remainder), `DIV` (Integer Division).
  - **Comparison**: `=`, `>`, `<`, `>=`, `<=`, `<>` (Not Equal).
  - **Boolean**: `AND` (Both true), `OR` (One true), `NOT` (Invert).
