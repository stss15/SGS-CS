# 3.4 Network Hardware - Revision Artifact Prompts

## Concept: Network Interface Card (NIC) & MAC Address
- **Type**: Hardware & Data Visual
- **Description**: The hardware needed to connect and its unique ID.
- **Prompt**: Create a visual for "NIC and MAC Address".
  - **Visual**: A green circuit board card (NIC) or a chip.
  - **Label**: "Network Interface Card. Allows device to connect to network."
  - **MAC Address Breakdown**: Show a hex code `00:1A:2B:3C:4D:5E`.
    - **First Half**: Label "Manufacturer ID (OUI)".
    - **Second Half**: Label "Device Serial Number".
    - **Note**: "Static. Assigned at factory. Unique to device."

## Concept: IP Address (IPv4 vs IPv6)
- **Type**: Comparison Visual
- **Description**: The logical address for the internet.
- **Prompt**: Create a comparison of "IPv4 vs IPv6".
  - **IPv4**:
    - Format: `192.168.1.1` (4 groups of denary).
    - Bits: "32-bit address."
    - Limit: "Running out of addresses."
  - **IPv6**:
    - Format: `2001:0db8:85a3:0000:0000:8a2e:0370:7334` (8 groups of hex).
    - Bits: "128-bit address."
    - Limit: "Virtually unlimited."

## Concept: Static vs Dynamic IP
- **Type**: Flow/Process Diagram
- **Description**: How IP addresses are assigned.
- **Prompt**: Create a diagram comparing "Static vs Dynamic IP".
  - **Dynamic IP**:
    - Visual: Device connecting to Router. Router asking DHCP Server.
    - Text: "Assigned by DHCP server. Changes on reconnection. Used for home devices. More private."
  - **Static IP**:
    - Visual: Server connecting to Internet.
    - Text: "Fixed address. Does not change. Used for Web Servers/Printers. Easier to track."

## Concept: The Router
- **Type**: Network Diagram
- **Description**: The role of a router in connecting networks.
- **Prompt**: Create a diagram showing "The Role of a Router".
  - **Central Icon**: Router.
  - **Connections**:
    - Left side: "LAN (Local Area Network)" with multiple computers.
    - Right side: "WAN / Internet".
  - **Functions**:
    - "Connects different networks."
    - "Routes data packets to correct destination."
    - "Assigns local IP addresses."
    - "Contains Firewall."

## Concept: MAC vs IP Address
- **Type**: Comparison Table
- **Description**: Physical vs Logical addressing.
- **Prompt**: Create a comparison table for "MAC vs IP".
  - **MAC Address**:
    - "Physical Address."
    - "Fixed / Static (Hardware)."
    - "Used within LAN (Switching)."
    - "Format: Hexadecimal (48-bit)."
  - **IP Address**:
    - "Logical Address."
    - "Changeable / Dynamic (Software)."
    - "Used for Routing across Internet."
    - "Format: Denary (IPv4) or Hex (IPv6)."
