# 2.3 Encryption - Revision Artifact Prompts

## Concept: Symmetric Encryption
- **Type**: Process Diagram
- **Description**: Visualizing the single-key process of symmetric encryption.
- **Prompt**: Create a linear process diagram.
  - **Step 1**: "Plaintext (Hello)".
  - **Step 2**: "Encryption Algorithm" + "Secret Key (Gold)".
  - **Step 3**: "Ciphertext (Xy7#b)".
  - **Step 4**: "Decryption Algorithm" + "Secret Key (Gold)".
  - **Step 5**: "Plaintext (Hello)".
  - **Warning Box**: "Key Distribution Problem: If the key is intercepted, the data is compromised."

## Concept: Asymmetric Encryption
- **Type**: Process Diagram
- **Description**: Visualizing the two-key process of asymmetric encryption.
- **Prompt**: Create a linear process diagram.
  - **Step 1**: "Plaintext (Hello)".
  - **Step 2**: "Encrypt with Receiver's PUBLIC Key (Green)".
  - **Step 3**: "Ciphertext (Xy7#b)".
  - **Step 4**: "Decrypt with Receiver's PRIVATE Key (Red)".
  - **Step 5**: "Plaintext (Hello)".
  - **Note Box**: "Public Key is shared with everyone. Private Key is kept secret."

## Concept: Encryption Scenario (Tom & Jane)
- **Type**: Storyboard / Comic Strip
- **Description**: A step-by-step visual story explaining how secure communication works.
- **Prompt**: Create a 5-step storyboard.
  - **Panel 1**: "Jane generates a Key Pair (Public & Private)".
  - **Panel 2**: "Jane sends her Public Key to Tom".
  - **Panel 3**: "Tom encrypts a document using Jane's Public Key".
  - **Panel 4**: "Tom sends the encrypted file to Jane".
  - **Panel 5**: "Jane decrypts the file using her Private Key".
  - **Style**: Simple characters, clear arrows, and document icons.

## Concept: Encryption Terminology
- **Type**: Visual Dictionary
- **Description**: Defining key terms with icons.
- **Prompt**: Create a visual glossary.
  - **Plaintext**: Icon of a readable document. "Original message."
  - **Ciphertext**: Icon of a scrambled/locked document. "Encrypted message."
  - **Encryption Algorithm**: Icon of a gear/processor. "The formula used to scramble data."
  - **Key**: Icon of a key. "The secret value used by the algorithm."
