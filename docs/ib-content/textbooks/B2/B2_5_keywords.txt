File processing: The manipulation of data stored in a file, commonly including reading from, writing to, and appending data.

Persistence: The characteristic of state that outlives the process that created it; data stored in a file remains available after the program terminates.

Text file: A sequence of lines of electronic text that is stored on a computer file system.

Parallel arrays: A data structure where two or more arrays of the same size are used to store related information at the same index.

Writing: The process of saving data from a program's memory to a file storage output stream.

Reading: The process of retrieving data from a file storage input stream into a program's memory.

Append: To add new data to the end of an existing file without overwriting its current contents.

Open mode: The specific access permission ('r' for read, 'w' for write, 'a' for append) used when opening a file.

Buffer: A temporary memory area used to move data between the program and the file system.

CSV (Comma Separated Values): A common text file format where data in lines is separated by commas.
