# Topic 5 End of Unit Test Plan

## Goal
Create a comprehensive 10-question test for Topic 5 (The Internet and its uses).

## Content Coverage
-   5.1 Internet & WWW
-   5.2 Digital Currency
-   5.3 Cyber Security

## Question Design

1.  **Internet vs WWW (Drag & Drop)**
    -   Sort features into Internet (Infrastructure) vs WWW (Service).

2.  **URL Components (Matching)**
    -   Match Protocol, Domain, Path, File to examples.

3.  **HTTP vs HTTPS (Dropdown)**
    -   Gap fill on security, encryption, and the padlock symbol.

4.  **Digital Currency (True/False)**
    -   Statements about Fiat vs Digital currency and centralisation.

5.  **Blockchain Process (Ordering)**
    -   Order the steps of adding a block (Transaction -> Hash -> Verify -> Add).

6.  **Malware Identification (Matching)**
    -   Match Virus, Worm, Trojan, Spyware to definitions.

7.  **Phishing vs Pharming (Categorisation)**
    -   Sort features (Email vs Code, Link vs Redirect).

8.  **Cyber Threats (Dropdown)**
    -   Gap fill for DDoS, Brute Force, Data Interception.

9.  **Security Measures (Matching)**
    -   Match Biometrics, Firewalls, Encryption, Anti-virus to functions.

10. **Social Engineering (Multiple Choice)**
    -   Identify the definition of Phishing or Pretexting.
