## Question Design

### Q1: Internet vs WWW (Categorisation)
**Concept**: Differences between the infrastructure and the service.
-   **Buckets**: Internet, WWW
-   **Items**:
    -   Worldwide collection of networks (Internet)
    -   Physical infrastructure (Internet)
    -   Uses TCP/IP (Internet)
    -   Collection of web pages (WWW)
    -   Accessed via browsers (WWW)
    -   Uses HTTP/HTML (WWW)

### Q2: URL Structure (Matching)
**Concept**: Parts of a Uniform Resource Locator.
-   **Pairs**:
    -   https -> Protocol
    -   www.google.com -> Domain Name
    -   /images/logo.png -> Path/Filename
    -   .com -> Domain Type

### Q3: HTTP vs HTTPS (Dropdown Gap Fill)
**Concept**: Secure data transfer.
-   **Text**:
    -   HTTP stands for (Hypertext Transfer Protocol).
    -   HTTPS uses (SSL/TLS) encryption for security.
    -   A (padlock) icon appears in the browser bar for HTTPS sites.
    -   HTTPS is essential for (online banking) and shopping.

### Q4: DNS Process (Ordering)
**Concept**: How a URL is converted to an IP.
-   **Order**:
    1.  User types URL into browser
    2.  Browser asks DNS server for IP address
    3.  DNS server finds IP matching the domain
    4.  DNS sends IP address back to browser
    5.  Browser sends request to the web server IP

### Q5: Cookies (Matching)
**Concept**: Session vs Persistent cookies.
-   **Pairs**:
    -   Session Cookie -> Stored in temporary memory, deleted when browser closes
    -   Persistent Cookie -> Stored on hard drive, remembers login details
    -   Tracking -> Using cookies to monitor user browsing habits
    -   Encryption -> Used to protect personal data in cookies

### Q6: Web Browsers (True/False Grid)
**Concept**: Functions of a browser.
-   **Statements**:
    -   Browsers interpret HTML to display web pages. (True)
    -   Browsers store the actual website files. (False - Servers do)
    -   History keeps a log of visited sites. (True)
    -   Bookmarks allow quick access to favourite sites. (True)
