# Topic 6 End of Unit Test Plan

## Goal
Create a comprehensive self-marking end-of-unit test for Topic 6 (Automated and Emerging Technologies).

## Content Coverage
-   6.1 Automated Systems
-   6.2 Robotics
-   6.3 Artificial Intelligence

## Question Design (10 Questions)

### Q1: Sensors vs Actuators (Categorisation) - *From 6.1*
**Concept**: Input vs Output devices in automation.
-   **Buckets**: Sensors, Actuators
-   **Items**: Temperature Sensor, pH Sensor, Motor, Heater, Light Sensor, Pump.

### Q2: The Automation Loop (Ordering) - *From 6.1*
**Concept**: The cycle of automation.
-   **Order**: Sensors take reading -> Data sent to Microprocessor -> Data compared to pre-set values -> Signal sent to Actuator -> Actuator alters physical condition.

### Q3: Robot Characteristics (True/False Grid) - *From 6.2*
**Concept**: Defining traits of a robot.
-   **Statements**: Ability to sense surroundings (True), Ability to move (True), Programmable (True), Must be humanoid (False).

### Q4: Independent vs Dependent Robots (Drag & Drop) - *From 6.2*
**Concept**: Human intervention levels.
-   **Buckets**: Independent, Dependent
-   **Items**: Mars Rover (Ind), Autonomous Vacuum (Ind), Surgical Robot (Dep), Factory Arm with Operator (Dep).

### Q5: Robot Components (Matching) - *From 6.2*
**Concept**: Functions of parts.
-   **Pairs**: Sensor -> Input, Microprocessor -> Brain, Actuator -> Movement, End-effector -> Tool.

### Q6: AI Categories (Matching) - *From 6.3*
**Concept**: Levels of AI.
-   **Pairs**: Narrow AI -> One task superior, General AI -> Human level, Strong AI -> Superior in many tasks.

### Q7: Expert System Components (Matching) - *From 6.3*
**Concept**: Parts of an Expert System.
-   **Pairs**: Knowledge Base -> Facts, Rules Base -> Logic, Inference Engine -> Processing, User Interface -> Interaction.

### Q8: AI vs Machine Learning (Categorisation) - *From 6.3*
**Concept**: Distinguishing fields.
-   **Buckets**: AI, Machine Learning
-   **Items**: Simulates human intelligence (AI), Uses training data (ML), Makes predictions (ML).

### Q9: Expert System Process (Ordering) - *From 6.3*
**Concept**: Logic flow.
-   **Order**: User Input -> Inference Engine -> Knowledge Base -> Rules Base -> Conclusion.

### Q10: Terminology (Multiple Choice) - *Mixed*
**Concept**: Key terms.
-   **Question**: Which of these is NOT a characteristic of an automated system?
-   **Options**:
    -   It requires constant human supervision (Correct)
    -   It uses sensors to monitor the environment
    -   It uses a microprocessor to process data
    -   It uses actuators to control equipment
