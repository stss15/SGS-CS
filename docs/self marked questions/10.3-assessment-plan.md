## Question Design

### Q1: Problem Statement to Logic (Multiple Choice)
**Concept**: Converting text to logic.
-   **Scenario**: "Alarm (X) sounds if Switch A is ON AND Switch B is OFF, OR if Switch C is ON."
-   **Question**: Select the correct logic expression.
-   **Options**:
    -   (A AND NOT B) OR C (Correct)
    -   (A OR NOT B) AND C
    -   A AND (NOT B OR C)
    -   (A AND B) OR C

### Q2: Logic to Truth Table (Ordering)
**Concept**: Solving `(A OR B) AND C`.
-   **Task**: Order the steps to find the output when A=0, B=1, C=1.
-   **Steps**:
    1.  Evaluate (A OR B): 0 OR 1 = 1
    2.  Evaluate C: 1
    3.  Evaluate (Result1 AND C): 1 AND 1
    4.  Final Output: 1

### Q3: Circuit Structure (Matching)
**Concept**: Identifying circuit patterns.
-   **Pairs**:
    -   (A AND B) OR C -> Two inputs to AND, result OR'd with C
    -   A AND (B OR C) -> B and C to OR, result AND'd with A
    -   NOT (A AND B) -> NAND gate equivalent
    -   NOT (A OR B) -> NOR gate equivalent

### Q4: Intermediate Values (Multiple Choice)
**Concept**: Tracing logic.
-   **Expression**: `X = (A AND B) XOR C`
-   **Inputs**: A=1, B=1, C=1.
-   **Question**: What is the final output X?
-   **Calculation**: (1 AND 1) = 1. 1 XOR 1 = 0.
-   **Options**: 0 (Correct), 1, Error.

### Q5: Logic Construction (Drag & Drop)
**Concept**: Building expressions.
-   **Task**: Construct "Output is 1 if A is NOT 1 AND B is 1".
-   **Buckets**: Expression Parts
-   **Items**: NOT A, AND, B.
