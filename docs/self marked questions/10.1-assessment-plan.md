## Question Design

### Q1: Gate Symbols (Matching)
**Concept**: Identifying gates by shape description.
-   **Pairs**:
    -   AND -> D-shaped
    -   OR -> Curved D-shaped
    -   NOT -> Triangle with circle
    -   NAND -> D-shaped with circle
    -   XOR -> Curved D-shaped with extra line

### Q2: AND Truth Table (Ordering)
**Concept**: Logic of AND gate.
-   **Order**:
    1.  0, 0 -> 0
    2.  0, 1 -> 0
    3.  1, 0 -> 0
    4.  1, 1 -> 1

### Q3: Gate Logic - AND (Multiple Choice)
**Concept**: Function of AND gate.
-   **Question**: Which gate outputs 1 ONLY if both inputs are 1?
-   **Options**: AND (Correct), OR, XOR, NAND.

### Q4: Gate Logic - XOR (Multiple Choice)
**Concept**: Function of XOR gate.
-   **Question**: Which gate outputs 1 if inputs are different?
-   **Options**: XOR (Correct), OR, AND, NOR.

### Q5: Gate Logic - NOT (Multiple Choice)
**Concept**: Function of NOT gate.
-   **Question**: Which gate has only one input and inverts it?
-   **Options**: NOT (Correct), AND, OR, XOR.
