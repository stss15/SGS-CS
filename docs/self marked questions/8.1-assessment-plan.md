## Question Design

### Q1: Variable vs Constant (Categorisation)
**Concept**: Difference between variables and constants.
-   **Buckets**: Variable, Constant
-   **Items**:
    -   Value can change during execution (Variable)
    -   Named data store for a fixed value (Constant)
    -   `PI = 3.142` (Constant)
    -   `Score = Score + 1` (Variable)
    -   `TaxRate = 0.2` (Constant - usually)
    -   `PlayerName` (Variable)

### Q2: Data Types (Matching)
**Concept**: Matching data types to examples.
-   **Pairs**:
    -   Integer -> 42
    -   Real -> 3.14
    -   Char -> 'A'
    -   String -> "Hello World"
    -   Boolean -> TRUE

### Q3: Input/Output (Ordering)
**Concept**: Sequence of a simple input/output program.
-   **Order**:
    1.  OUTPUT "Enter your name"
    2.  INPUT Name
    3.  OUTPUT "Enter your age"
    4.  INPUT Age
    5.  OUTPUT "Hello " + Name + ", you are " + Age

### Q4: Loop Types (Multiple Choice)
**Concept**: Identifying the correct loop for a scenario.
-   **Scenario**: You want to print "Hello" exactly 10 times.
-   **Options**:
    -   FOR Loop (Correct)
    -   WHILE Loop
    -   REPEAT...UNTIL Loop
    -   IF Statement

### Q5: Operator Logic (Multiple Choice)
**Concept**: Understanding logical operators.
-   **Snippet**: `IF (Age > 18) AND (HasLicense = TRUE)`
-   **Question**: When will this condition be TRUE?
-   **Options**:
    -   Only if both Age is > 18 AND HasLicense is TRUE (Correct)
    -   If Age is > 18 OR HasLicense is TRUE
    -   If Age is 18 AND HasLicense is FALSE
    -   If Age is < 18
