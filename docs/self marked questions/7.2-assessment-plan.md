Create a self-marking assessment for Topic 7.2 using the standard template.

## Question Design

### Q1: Flowchart Symbols (Matching)
**Concept**: Identifying standard symbols.
-   **Pairs**:
    -   Oval -> Terminator (Start/Stop)
    -   Rectangle -> Process (Action)
    -   Parallelogram -> Input/Output
    -   Diamond -> Decision (Yes/No)

### Q2: Pseudocode Loops (Categorisation)
**Concept**: Characteristics of different loops.
-   **Buckets**: FOR Loop, REPEAT Loop, WHILE Loop
-   **Items**:
    -   Count-controlled (FOR)
    -   Fixed number of iterations (FOR)
    -   Post-condition check (REPEAT)
    -   Runs at least once (REPEAT)
    -   Pre-condition check (WHILE)
    -   May never run (WHILE)

### Q3: Decomposition Terms (Matching)
**Concept**: Key definitions.
-   **Pairs**:
    -   Top-down design -> Breaking a system into sub-systems
    -   Stepwise refinement -> Breaking sub-systems into smaller parts
    -   Structure Diagram -> Hierarchical diagram of the system
    -   Algorithm -> Sequence of steps to solve a problem

### Q4: System Components (Drag & Drop)
**Concept**: Parts of a computer system.
-   **Buckets**: Input, Process, Output, Storage
-   **Items**:
    -   Keyboard (Input)
    -   Sensor Data (Input)
    -   Calculation (Process)
    -   Comparison (Process)
    -   Screen Display (Output)
    -   Saved File (Storage)

### Q5: Pseudocode Operators (Matching)
**Concept**: Understanding syntax.
-   **Pairs**:
    -   `<-` -> Assignment
    -   `<>` -> Not Equal
    -   `^` -> Power of
    -   `DIV` -> Integer Division (Note: Text uses `/` for divide, but `DIV` is common in IGCSE. Sticking to text: `*` -> Multiply)

### Q6: Conditional Logic (Ordering)
**Concept**: Structure of an IF statement.
-   **Order**:
    1.  IF Age < 18
    2.  THEN
    3.  OUTPUT "Child"
    4.  ELSE
    5.  OUTPUT "Adult"
    6.  ENDIF
