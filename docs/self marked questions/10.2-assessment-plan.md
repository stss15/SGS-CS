## Question Design

### Q1: NAND Gate Logic (Multiple Choice)
**Concept**: Function of NAND gate.
-   **Question**: Which gate outputs 0 ONLY if both inputs are 1?
-   **Options**: NAND (Correct), AND, NOR, XOR.

### Q2: NOR Truth Table (Ordering)
**Concept**: Logic of NOR gate.
-   **Order**:
    1.  0, 0 -> 1
    2.  0, 1 -> 0
    3.  1, 0 -> 0
    4.  1, 1 -> 0

### Q3: Logic Notation (Matching)
**Concept**: Matching gate names to logic notation.
-   **Pairs**:
    -   AND -> A AND B
    -   OR -> A OR B
    -   NOT -> NOT A
    -   NAND -> A NAND B
    -   NOR -> A NOR B

### Q4: XOR Gate Properties (True/False Grid)
**Concept**: Understanding XOR behavior.
-   **Statements**:
    -   Output is 1 if inputs are different (True)
    -   Output is 1 if inputs are same (False)
    -   Output is 0 if both inputs are 1 (True)

### Q5: Simple Logic Calculation (Multiple Choice)
**Concept**: Applying logic.
-   **Question**: If A = 1 and B = 0, what is the output of (A OR B)?
-   **Options**: 1 (Correct), 0, Error, Null.
